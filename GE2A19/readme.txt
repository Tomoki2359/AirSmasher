操作方法
・マウスを動かす…マレットがその方向に動く
・左クリック…マレットをステージにつける/パックを止める/(タイミングよく落とせば)/パックを置く
・右クリック…カメラの視点変更

説明
・ゲーム内容はエアホッケーになっております。
・右上の数字は点数になっており、左が相手、右が自分の点数になっております。
・パックがゴールした時はパックがゴールしたほうにパックがおけるようにしております。
・ゴール判定は壁にぶつかった際にパックの進む少し先の地面があるかどうかで判断しております。
・すり抜け防止、めり込み分戻す処理をしております。
・現在は敵を製作中です。

ゴール方向にパックを動かす方法
・マレットにぶつかったときのパックの動きの逆の順番でやっていけば求められるのでは？と考えて製作しております。
・マレットの位置はぶつかると予定している位置からパックの方向分マレットとパックの半径をずらした位置に
移動してパックがゴールの方向に向かうように動かそうとしています。
